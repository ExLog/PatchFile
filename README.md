A GUI to generate patch files such as 
- Patch[version].txt
- Patch[version].pak.md5
- PatchInfoServer.cfg

by dragging your PAK file onto window.

## Configuration

Put your patch path into patch_path.txt.

e.g.
```
C:/xampp/htdocs/Patch/
```

Or if you use php 5.2
```
C:/php5.2/WWW/GX/
```
